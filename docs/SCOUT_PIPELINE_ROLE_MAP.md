# HEHA Scout Pipeline role map

| Role | Dashboard lens | Scout leads | Event leads | Public approval |
|---|---|---|---|---|
| `super_admin` | Admin, PM, SOM, Events | Full internal access | Full internal access | Existing admin approval flow |
| `developer_admin` | Admin, PM, SOM, Events | Full internal access | Full internal access | Technical/admin support |
| `pm_admin` | Project Manager | Read/write | Read existing event context | No automatic publish |
| `som_admin` | Sales Operations | Create/update/handoff | Create/update/handoff | No automatic publish |
| `community_events_admin` | Events | Shared read access | Event lead create/update | No automatic publish |

The Scout Pipeline is a shared data module. Dashboard lenses filter workflow context; they are not separate CRMs.
