# Scout Pipeline UI notes

The internal workspace is intentionally mounted at `/internal` instead of adding internal controls to the customer navigation.

Admin users may switch between Admin, Project Manager, Sales Operations, and Events lenses. Non-admin internal users see only the lens allowed by their active `user_roles` assignment.

The intake form is mobile-first for field visits. Business name and contact fields remain optional so a team member can capture a location or idea immediately and enrich it later.

Event lead types reveal venue and event intelligence fields. Public partner visibility continues to use the existing approval-gated partner status flow.
